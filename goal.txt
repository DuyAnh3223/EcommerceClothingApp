1/Shipping (Vận chuyển):
So với schema bookstore (ảnh bạn gửi), app quần áo hiện chưa có hệ thống quản lý vận chuyển như:

shipping_provider

shipping

order_tracking

shipper

shipping_evidence

➡️ Gợi ý:

sql
Copy
Edit
-- Thêm bảng shipping_provider nếu dùng nhiều đơn vị giao hàng
CREATE TABLE shipping_provider (
  id INT PRIMARY KEY AUTO_INCREMENT,
  name VARCHAR(100),
  contact_info VARCHAR(255),
  created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Quản lý vận chuyển cho từng đơn hàng
CREATE TABLE shipping (
  id INT PRIMARY KEY AUTO_INCREMENT,
  order_id INT NOT NULL,
  provider_id INT,
  tracking_number VARCHAR(100),
  estimated_delivery DATE,
  delivered_at DATE,
  status VARCHAR(50),
  FOREIGN KEY (order_id) REFERENCES orders(id) ON DELETE CASCADE,
  FOREIGN KEY (provider_id) REFERENCES shipping_provider(id)
);
2. Mã giảm giá / voucher:
Hiện chưa thấy bảng voucher như trong bookstore.
➡️ Nếu muốn triển khai khuyến mãi, bạn cần có:

sql
Copy
Edit
CREATE TABLE voucher (
  id INT PRIMARY KEY AUTO_INCREMENT,
  code VARCHAR(50) UNIQUE,
  description TEXT,
  discount_type ENUM('percent', 'amount'),
  discount_value DECIMAL(10,2),
  min_order_value DECIMAL(10,2),
  max_discount DECIMAL(10,2),
  quantity INT,
  valid_from DATE,
  valid_to DATE
);
→ Và liên kết vào orders.
